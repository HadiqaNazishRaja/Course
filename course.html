👨‍🏫Web Languages

HTML, CSS and JavaScript are the three most common programming languages that are used to build web pages and apps. Each language has it's own importance and using them all togeather allows building great web pages and apps that not only look great but are also user interactive. HTML is the basic language utilized to form the basis of any web page. Other coding languages can be integrated into HTML files to modify the look of the website.

HTML

HTML arranges the content of a web page. It designs the basic structure of the contents ( text, images videos etc) of a webpage. It is actually the backbone of every webpage.HTML is a markup language that we use to describe its content and layout a basic rough structure of a website.

CSS

CSS is a styling language that gives styling of a website. With the help of front color, background color and border styling CSS furnishes the look of the site. CSS is a styling language that we use to shape and style website content in terms of colors, typography, spacing, borders, and more. It is applied to HTML.

JavaScript

JavaScript is used to create interaction between webpages and the user. JavaScript is a dynamic programming language that allows you to make websites interactive, and reactive. It creates the interactive webpage that gives life to webpage by inteacting with the user like to click button, alert to pop up. It is applied to HTML.


📚HTML: HyperText Markup Language

HTML is the abbreviation of HyperText Markup Language. HTML is the basic language of each website. It creates the basic structure of the webpage.It forms the basis of all web pages. It forms the basic structural content of any web page. It is necessary for the basic formatting, arrangement and display of the contents of any web page. So, in short, HTML is used to control the following basic things of web content:

⌨️How to create the Webpage Structure by using HTML

HTML document is based on mainly three components:

HTML Tags HTML Element HTML Attribute
Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags. Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag. It is used to define the character of an HTML element. It always placed in the opening tag of an element. It generally provides additional styling (attribute) to the element.
HTML tags are used to hold the HTML element. HTML element holds the content. HTML attributes are used to describe the characteristic of an HTML element in detail.
HTML tag starts with < and ends with > Whatever written within a HTML tag are HTML elements. HTML attributes are found only in the starting tag.
HTML tags are almost like keywords where every single tag has unique meaning. HTML elements specifies the general content. HTML attributes specify various additional properties to the existing HTML element.


🚀HTML Tags

HTML document is made up of individual HTML elements. Elements are created using opening and closing tags. Each tag is the name of element enclosed in angle brackets (<>). The content goes between tags. Each HTML tag is instruction to browser, telling it how to show content on screen. Tags allow you to add text, audio, videos, images and paragraphs to webpage and divide information into sections, such as line and paragraphs.

Opening Tags

Opening tags are generated by enclosing the symbol of the desired element in angle brackets <> at the start of the content. The content of the elements are enclosed within the opening and closing tags, so, opening tag encloses the content in the start.

Closing tags

Closing tags are generated by enclosing the symbol of the desired element at the end of the contents. The content of the elements are enclosed within the opening and closing tags, so, closing tag closes the content at the end.


🐛HTML Elements

Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag. HTML document is made up of individual HTML elements. There are basically two types of HTML elements

  1. Block Elements
  2. Inline Elements
Inline Elements Block Elements
Inline elements occupy only sufficient width required. Block Elements occupy the full width irrespective of their sufficiency.
Inline elements don’t start in a new line. Block elements always start in a line.
Inline elements allow other inline elements to sit behind. Block elements doesn’t allow other elements to sit behind
Inline elements don’t have top and bottom margin Block elements have top and bottom margin.

🔗How to link other languages (CSS/JS) within HTML document

HTML, CSS and JavaScript work together to form the front-end design of a website by applying information that affects content, style and interactivity of a site. HTML creats the content of website, CSS styles the website and JS makes the website interactive.


How to link to CSS

In order to make use of the CSS capabilities it needs to be linked within the HTML content so that style can be added to the website. CSS will tell the browser how to display the existing HTML. CSS can be compared to adding personal style to the body. When you link CSS to HTML, it’s like dressing up the body. There are two ways of incorporating CSS within the html document

  • Introducing opening and closing tags of style to introduce CSS within head of html document


  • Introducing link of a seprately genetared css document in the head of html document


How to link to JS

JS controls the behaviour of the website and creates the interactive aspect of website that allows the user to interact within the browser. There are two ways of introducing javascript in html file.

  • Introducing opening and closing tags of script to introduce JS at the end before the closing tag of body of html document


  • Introducing link of a seprately genetared js document at the end before the closing tag of body of html document
